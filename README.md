# ローグライク - ダンジョン探索

ランダム生成のダンジョンを探索するローグライクゲームです。

## 遊び方

1. `index.html` をブラウザで開く
2. 「冒険を始める」をクリック
3. **矢印キー**（← ↑ → ↓）でプレイヤー（@）を移動
4. 敵（E）に隣接すると自動で攻撃
5. 階段（>）を見つけて次の階へ進む
6. 敵を倒しながら深い階層を目指そう！

## 特徴

- **ランダムダンジョン生成**: 部屋と廊下を組み合わせたプロシージャル生成
- **視界システム**: 壁越しは見えない、探索した場所は薄く表示
- **敵AI**: プレイヤーを追いかけて攻撃
- **階層システム**: 深くなるほど敵が強くなる

## ファイル構成

- `index.html` - メインHTML
- `style.css` - スタイル（ダークテーマ）
- `game.js` - ゲームロジック
